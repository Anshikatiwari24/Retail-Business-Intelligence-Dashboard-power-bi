# 🛒 Retail Business Intelligence Dashboard

## 📌 Project Overview

The Retail Business Intelligence Dashboard is an end-to-end Power BI solution designed to transform retail transaction data into actionable business insights.

This project enables stakeholders to monitor sales performance, profitability, customer behavior, regional trends, and future sales forecasts through interactive visualizations and advanced analytics.

The dashboard was built using Power BI, Power Query, DAX, and Star Schema data modeling principles to support data-driven decision-making.

---

# 🎯 Business Problem

Retail organizations generate large volumes of transactional data every day.

Without a centralized reporting system, it becomes difficult to:

- Track sales performance
- Monitor profitability
- Identify top-performing products
- Understand customer behavior
- Analyze regional performance
- Compare actual performance against targets
- Forecast future business trends

This dashboard addresses these challenges by providing a unified business intelligence solution.

- Repeat customer percentage remained consistently high.
- Regional performance varied significantly across markets.
- Forecasting highlighted future growth opportunities.
